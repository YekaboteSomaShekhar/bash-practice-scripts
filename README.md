### Overview of the Project

This repository contains a collection of simple Bash scripting files such as operators, Counting the number of files in the current directory, Temperature converter and understand basic shell scripting concepts such as loops, conditions, arithmetic operations, and user input.

### Tasks to do
1. Write a script that asks for two numbers and operators(+,-,*) and performs the calculation.
2.  Write a script that counts how many files are in the current directory.
3. Write a script that asks for multiple names(in a loop) and greets each person, stopping when the user types "done".
4. Temperature converter: Convert celsius to fahrenheit (formula: F = C * 9/5 +32).

### Commands I Used

+ read -p → prompts the user to enter a name.
+ ls -1 → lists all files and directories.
+ wc -l → counts how many files are in the directory.
+ echo → prints the text.
